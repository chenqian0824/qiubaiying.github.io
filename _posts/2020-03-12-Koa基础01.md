## Koa脚手架


1.安装方法： 

	npm install -g koa-generator

2.创建项目

	koa2 -e projectName

3.项目初始化

	cd projectName && npm install

4.运行项目
	
	// npm run dev 开发模式，支持热更新
	SET DEBUG=koa* & npm start koa2-lean | npm run dev
